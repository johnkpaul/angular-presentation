TypeScript

TypeScript is a language that compiles to JavaScript

  • Created by Anders Hejlsberg, lead architect of Turbo Pascal, Delphi and later C#
  • Rather than try to replace JavaScript, TypeScript embraces and extends it

TypeScript

  • TypeScript is Based on EcmaScript 5 (ES5) which is the current version of JavaScript standard, supported in most of the modern browsers.
  • Adds features from the next JavaScript version (ES6), which is not supported well by other browsers otherwise.
  • On top of it adds an optional Type System and some other features like decorators (we'll learn about them later)

Classes

Angular makes heavy use of TypeScript classes

They work similarly to classes in other languages. They group together methods and properties of an object.

Classes - constructor

There's a special method on a class called constructor. It's run when the class is instantiated and allows the class to take parameters.

Classes - Access modifiers

Constructor parameters marked as public (or private, or protected) become a class property accessible as this.name

private or protected properties are not visible outside of the class.

Classes - export

By the way, did you notice the export keyword before class? It is used to share information between files. Next slide will show how to export and use this class in a different file.
